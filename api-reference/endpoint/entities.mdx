---
title: "Entities"
api: "GET https://edb.xeol.io/v1/entities"
description: "This endpoint returns a list of supported entities."
---

### Body

<ParamField body="type" type="string">
  The type of entity to return. Can be one of `software`, `package`, `os`, `device`, `service`.
</ParamField>

### Response

<ResponseField name="entities" type="object[]">
 <Expandable title="entity">
    <ResponseField name="name" type="string">
      The name of the entity
    </ResponseField>
    <ResponseField name="vendor" type="string">
      The vendor of the entity
    </ResponseField>
    <ResponseField name="type" type="string">
      The type of the entity
    </ResponseField>
  </Expandable>
</ResponseField>


<RequestExample>
  ```bash curl
  curl --location --request GET 'https://edb.xeol.io/v1/entities' \
  ```
</RequestExample>

<ResponseExample>
  ```json 200
  [
    {
        "name": ".NET",
        "vendor": "Red Hat",
        "type": "software"
    },
    {
        "name": ".NET",
        "vendor": "Microsoft",
        "type": "software"
    }
  ]
  ```
  ```json 400
  {
    "message": "Invalid entity type. Valid types are: [service device software package os]"
  }
  ```
</ResponseExample>
